# Autonomous-Robot-Platoon-System
This repository contains python code for autonomous navigation of mobile robots. 

Robots   : Pioneer 3DX
LIDAR    : SICK LMS200
Platform : ROS (Robot Operating System) Kinetic 
OS       : Ubuntu 16.04 LTS
Python   : 3.5.2
